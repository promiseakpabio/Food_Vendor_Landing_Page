# Mama T's Kitchen Landing Page

This is a clean and responsive landing page built for a food vendor business.

## Purpose
The goal of this project is to create a simple, professional website that helps a food business present its meals clearly and allows customers to easily place orders.

## Features
- Responsive design (works on mobile and desktop)
- Hero section with call-to-action
- About section
- Menu display
- “Why Choose Us” section
- Customer testimonial
- WhatsApp order button

## Built With
- HTML
- CSS (Flexbox + responsive design)
- JavaScript (for WhatsApp interaction)

## What I Learned
- Structuring a real business landing page
- Improving layout and spacing using CSS
- Making designs responsive for mobile users
- Creating simple but effective user interfaces
